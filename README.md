# Home-automation-and-safety-systems.
IoT based Home automation and safety systems.

#Components

-Arduino Uno.

-4 Channel Relay module.

-DHT11 sensor.

-MQ2 smoke sensor.

-Bluetooth Module HC-05.

-OLED display.
